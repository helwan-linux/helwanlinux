  • Rust, Go, Node.js, Python, PHP
  • Docker, Git, SQLite, MariaDB

🖥️ Desktop & System

  • Cinnamon Desktop, LightDM, Gedit
  • LibreOffice, GNOME Calculator, Screenshot tools

🎮 Games & Gaming Tools

  • Gamemode, vkd3d, Cuyo, Blurble
