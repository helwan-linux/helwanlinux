Helwan Stream v1.0

The Ultimate Resource-Saving Video Engine for Linux

What is Helwan Stream?

Helwan Stream is a lightweight desktop application designed to search, stream, and download online videos without opening a web browser. It acts as a high-performance "filter" that extracts only the video content you want, leaving behind the heavy scripts and ads that slow down your PC.

Key Features:

  • Minimal RAM Usage: Runs on only ~90MB of RAM compared to 1GB+ in modern browsers.
  • Built-in Adblock: Streams the raw video directly, bypassing all web ads and trackers.
  • High Performance: Uses mpv for ultra-light playback and aria2c for multi-threaded downloads.
  • Data Saver: Smart caching and flat-extraction save up to 50% of your internet bandwidth.

How to Install:

Open your terminal and use one of the following commands:

# Method 1: Using Helwan Package Manager (Recommended)
hpm i hel-stream

# Method 2: Using Standard Pacman
sudo pacman -S hel-stream
