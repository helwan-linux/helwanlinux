Welcome to Helwan Linux

Helwan Linux is a modern Arch-based Linux distro crafted for developers, power users, gamers, and open source lovers. It ships with essential tools and unique utilities built in-house to enhance productivity, security, and development experience.

🚀 Key Features

  • Arch Linux core with fast installation via Calamares
  • Preconfigured development tools and editors
  • Essential drivers and multimedia codecs
  • Privacy-respecting and community-driven
  • Dozens of custom-built Helwan apps (hel-* series)
  • Ready for gamers: Steam, Lutris, MangoHud, Gamemode

📦 Main Package Categories

  • Development: VS Code, QtCreator, Rust, Go, Node.js
  • Desktop: Cinnamon, LightDM, Gedit, LibreOffice
  • Games: Steam, OpenRA, Lutris
  • Multimedia: VLC, Audacious, Gimp, Cheese
  • Helwan Tools: hel-terminal, hel-builder, hel-text-editor, hel-tutorial
