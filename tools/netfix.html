Helwan Linux NetFix GUI - User Guide

Purpose

The Helwan NetFix GUI is a graphical tool for Arch-based systems designed to quickly perform maintenance and fixes. It helps users repair the package database, clear caches, fix corrupted packages, update the system, refresh mirrors, and run a full automatic fix.

Main Features

  • Repair Database: Removes unused packages.
  • Clear Cache: Deletes old cached packages.
  • Fix Corrupted Packages: Resolves overwrite or conflict errors.
  • Update System: Synchronizes packages with the latest versions.
  • Refresh Mirrors: Updates package server mirrors for speed and reliability.
  • Full Auto Fix: Runs all maintenance tasks in order automatically.
  • Live Output: Displays command output in a scrollable text area.
  • Progress Bar: Shows ongoing processes visually.

How It Works

The application uses PyQt5 for its graphical interface. Commands are executed using pkexec to request administrative privileges. The GUI updates in real-time with command output and progress.

Buttons & Commands

  • Each button corresponds to a specific system maintenance task.
  • Clicking a button runs the associated command.
  • While a command is running, the interface is disabled to prevent interference.
  • “Full Auto Fix” runs all commands sequentially, stopping if an error occurs.

Output & Progress

  • The text area shows live output from the commands.
  • Errors are clearly labeled with ERROR:.
  • The progress bar appears during command execution to indicate activity.

Running the Tool

  1. Open a terminal.
  2. Navigate to the folder containing hel_netfix_gui_qt.py.
  3. Run the tool with:
    4. python3 hel_netfix_gui_qt.py
Notes for Users:
  • Do not close the application while commands are running.
  • Ensure pkexec is installed for privilege escalation.
  • The tool is safe for Arch-based systems, but always keep backups of important data.
