Helwan Blocks

Welcome!

Helwan Blocks is a modern block-stacking puzzle game (like Tetris) for Helwan Linux. Your goal is to place falling blocks to complete horizontal lines and score points. Clear more lines at once to earn bigger bonuses!

Tip: Focus on planning ahead and keeping your stack flat for maximum score!

Main Menu

  • START GAME: Begin a new game immediately.
  • INSTRUCTIONS: Learn how to play Helwan Blocks and what each key does.
  • HIGH SCORES: View the top scores you or others have achieved.
  • EXIT: Close the game safely.

How to Play

Use the keyboard to control the falling blocks:

  • Left Arrow: Move the block left
  • Right Arrow: Move the block right
  • Down Arrow: Move the block down faster (soft drop)
  • Up Arrow: Rotate the block
  • Spacebar: Instantly drop the block to the bottom (hard drop)
  • C Key: Hold or swap the current block with a stored block
  • Clear full rows to score points. Clearing multiple rows gives extra points!
  • Press M anytime to return to the main menu.
Tip: Using the hold function strategically can help you set up combos!

Game Screen Overview

  • Score Panel: Shows your current score.
  • Next Block: Displays the upcoming block so you can plan ahead.
  • Held Block: Displays the block you have on hold.
  • Playing Field: The area where blocks fall and are stacked.
  • Game Over Screen: Appears when blocks reach the top. You can restart by pressing R or return to menu with M.

Scoring Tips

  • Clear 1 line: small points
  • Clear 2 lines: more points
  • Clear 3 lines: bigger bonus
  • Clear 4 lines at once (a Tetris!): maximum points
  • Combo bonus: clearing lines consecutively gives extra points.

Exiting the Game

  • From the main menu, click EXIT or press the window close button.
  • Game automatically saves your high scores locally.
