Momo - Helwan Linux Diagnostics Tool

Version: Stable (TUI with Curses and Persistent Config)
Author: Saeed Badrelden

Overview

Momo is a Linux-based **diagnostics tool** designed to help users quickly check the health and performance of their system, including RAM, CPU, storage, and network. It provides a **text-based interactive menu (TUI)** for easy navigation and offers persistent configuration for stress test durations.

The tool is designed for both **quick checks** and **deep diagnostics**, including:

  • RAM and CPU stress testing with configurable durations
  • Disk health and performance checks (including NVMe)
  • Network bandwidth and path quality testing
  • Automatic logging of all test outputs

Note: Some tests require root privileges (especially NVMe and disk tests). Running Momo with sudo is recommended for full functionality.

System Requirements

  • Linux operating system
  • Terminal with enough size (recommended ≥ 80x24)
  • Installed tools (some are optional):
    • stress-ng (RAM/CPU stress tests)
    • memtester (RAM endurance tests)
    • sysbench (RAM speed test)
    • smartctl (SMART disk tests)
    • fio (Disk IOPS test)
    • iperf3 (Network bandwidth)
    • mtr (Network path quality)
    • nvme-cli (NVMe diagnostics)

Launching Momo

Open a terminal and run:

python3 momo.py

Or, for full disk and NVMe functionality:

sudo python3 momo.py

Interactive Menu

Once launched, Momo presents a scrollable menu where you can select tests using the arrow keys and Enter. Additional commands:

  • A - Run all tests automatically
  • Q - Quit Momo
  • During a test, Q/S - Stop the test early
  • Scroll within long test outputs using ↑ and ↓ keys

Available Tests

  • RAM Usage: Displays current memory usage.
  • RAM Details: Shows detailed memory info from /proc/meminfo.
  • RAM Stress Test (Short): Quick RAM stress using stress-ng.
  • RAM Endurance Test (Long): Extended RAM stress test for endurance.
  • Memtester 512M (x5): Runs memory tests 5 times.
  • Memory Speed: Measures memory speed using sysbench.
  • Swap Usage: Shows active swap partitions.
  • CPU Info: Detailed CPU information.
  • CPU Stress Test: Stress test for CPU using stress-ng.
  • Smart Status: SMART health info for disks (requires smartctl).
  • Disk IOPS (Fio Test): Measures random I/O performance using fio.
  • Disk Usage: Shows filesystem usage (df -h).
  • Sensors: Displays CPU and motherboard temperatures.
  • Network Ping (Basic): Simple ping test to google.com.
  • Network Bandwidth (iperf3): Measures download/upload bandwidth.
  • Network Path Quality (mtr): Measures latency and packet loss along the network path.
  • NVMe Smart Info: SMART info for NVMe drives (requires nvme-cli and sudo).

Stress Test Configuration

You can configure the duration (in seconds) of stress tests through the menu option "Configure Stress Test Durations". Changes are saved automatically in a persistent configuration file (~/.momo/config.json).

Default Durations:

  • RAM Stress Test (Short): 30 seconds
  • RAM Endurance Test (Long): 3600 seconds (1 hour)
  • CPU Stress Test: 20 seconds

Logs

All test outputs are automatically logged in:

~/.momo/logs

Logs include timestamp, test name, and full command output for later review.

Tips for Users

  • Run Momo in a standard terminal with enough size to avoid display issues.
  • Root privileges are recommended for Disk and NVMe tests.
  • Long stress tests may take a significant amount of time depending on duration settings.
  • Network tests require an active internet connection.

Troubleshooting

  • Missing tool errors: Install the required tool using your package manager (e.g., pacman, apt, dnf).
  • Curses display issues: Ensure your terminal is large enough; resizing may adjust scrollable menus automatically.
  • Permission errors: Use sudo for tests that require access to hardware devices.

Summary

Momo is a **user-friendly, terminal-based diagnostic tool** designed to give both quick insights and deep system tests for Helwan Linux (and compatible Linux distributions). It is ideal for power users and technicians who want:

  • Real-time monitoring
  • Stress and endurance testing
  • Automated logging and historical review

With persistent configuration and scrolling UI, Momo ensures that both short and long-running tests are handled efficiently and safely.
