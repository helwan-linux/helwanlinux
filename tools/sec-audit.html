Starting a Scan

  1. Click the Start Scan button.
  2. The Scan Window opens and shows progress.
  3. The scan runs using a background thread so the main window stays active.
  4. Progress updates appear live in the Scan Window.
  5. When the scan finishes, the Results Window opens with the findings.
Note: Do not close the application while a scan is running; this may interrupt the process.

Scan Window

This window shows the current scan progress:

  • Progress bar with percentage completed.
  • Current check being performed.

Results Window

After the scan, this window displays:

  • List of security issues found.
  • Severity of each issue.
  • Recommendations to fix or improve security.

Settings Window

The settings window allows you to configure application-specific options, such as:

  • Scan preferences (which checks to enable/disable).
  • Notification settings.
  • Theme preferences.
