Overview

HPM is a human-friendly package manager for Helwan Linux, built on top of pacman with natural-language command chaining. Each language supports its native "then" keyword for executing multiple commands in a row.

Installation

On Helwan Linux:

sudo pacman -S hpm

On other Arch-based distributions:

yay -S hpm
