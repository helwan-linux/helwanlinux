Helwan Text Editor – User Guide

1. Overview

Helwan Text Editor is a simple and powerful text editor. You can type, edit, and format text easily.

  • Standard text editing (cut, copy, paste, undo, redo)
  • Find and replace text
  • Text formatting (bold, italic, underline, font size, font family, colors)
  • Text alignment and direction (Left-to-Right or Right-to-Left)
  • Word and character count

2. Main Window Layout

Text Area

  • Type your text here.
  • Words wrap automatically to the window width.
  • Placeholder text: Start typing here...

Menu Bar

File Menu

  • New (Ctrl+N): Start a new document
  • Open (Ctrl+O): Open an existing file
  • Save (Ctrl+S): Save the current file
  • Save As (Ctrl+Shift+S): Save with a new name
  • Cut / Copy / Paste: Standard editing actions
  • Select All (Ctrl+A): Select all text
  • Exit (Ctrl+Q): Close the editor

Edit Menu

  • Undo / Redo (Ctrl+Z / Ctrl+Y)
  • Find (Ctrl+F): Search for text
  • Replace (Ctrl+H): Find and replace text

Format Menu

  • Bold / Italic / Underline (Ctrl+B / Ctrl+I / Ctrl+U)
  • Font Color / Highlight Color: Change text or background color
  • Text Alignment: Left, Center, Right
  • Text Direction: Left-to-Right or Right-to-Left

Help Menu

  • About: View application information

Toolbar

  • Quick access buttons: New, Open, Save, Undo, Redo
  • Font Family: Choose a font from the list
  • Font Size: Choose text size
  • Find / Replace buttons

Status Bar

  • Shows Ready or other messages
  • Displays word count and character count

3. Quick Tips

  • To format text, select it first, then choose Bold, Italic, or Underline.
  • Use Find/Replace to quickly search or replace text in your document.
  • Check word/character count in the bottom bar.
  • Toolbar combos allow instant font and size changes.
Note: Keyboard shortcuts make editing faster. Try using Ctrl+S to save frequently!
