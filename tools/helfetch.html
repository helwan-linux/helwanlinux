Helfetch: System Information Fetcher

Helfetch is the enhanced, next-generation system information tool for Helwan Linux. It provides a beautiful, high-speed overview of your system's health, hardware, and environment directly in your terminal.

Why Use Helfetch?

  • Comprehensive Data: View everything from RAM state to network status in one go.
  • Visual Health Checks: It doesn't just show numbers; it analyzes your RAM state and colors the output accordingly.
  • Security Awareness: Includes system integrity and security status checks.
  • Inspirational: Features daily quotes to keep you motivated while you work.

How to Use

Basic Command

Open your terminal and type:

helfetch

Options

Command Description
--no-logo Display system info without the Helwan Linux ASCII logo.
--help Show the help message and exit.

Information Captured

The tool collects data from four core modules:

  1. System: OS version, security status, and system integrity.
  2. Hardware: CPU, RAM usage, and Disk health.
  3. Desktop: Desktop Environment (DE), Window Manager (WM), and themes.
  4. Network: Current connection status and IP info.

Troubleshooting

If the tool encounters an error, it will display a red error message. Common issues are usually related to missing permissions or corrupted configuration files. In case of a crash, you can safely exit using Ctrl+C.
