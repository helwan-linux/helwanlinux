Helfetch - Helwan Linux System Information Tool

Overview

Helfetch is a custom system information fetcher for Helwan Linux. It collects and displays comprehensive information about your system, hardware, desktop environment, and network. The tool can also show an inspirational quote and the Helwan Linux ASCII art logo.

Usage

Run Helfetch from the terminal:

python3 helfetch.py [options]

Available options:

  • --no-logo: Do not display the Helwan Linux ASCII art logo at the top.

Data Collected

Helfetch gathers data from multiple sources and modules:

  • System Info: OS version, kernel, security status, system integrity.
  • Hardware Info: RAM size and state, disk usage and performance.
  • Desktop Info: Desktop environment, window manager, screen resolution.
  • Network Info: Active connections, IP addresses, network interfaces.

All collected data is merged into a single structured output.

Output

The output is formatted in a human-readable style with optional colors for clarity:

  • ASCII Logo: Displays the Helwan Linux logo (optional).
  • Information Keys: Colored according to info_key_color setting.
  • Information Values: Colored according to info_value_color setting.
  • RAM State: Displayed with a colored bar indicating usage level.
  • Inspirational Quote: A motivational message included at the bottom.

Modules Overview

  • core.system_info: Retrieves OS details, security, and integrity status.
  • core.hardware_info: Retrieves RAM, disk, and general hardware stats.
  • core.desktop_info: Retrieves desktop environment and display info.
  • core.network_info: Retrieves network connection and interface info.
  • display.ascii_art: Provides Helwan Linux ASCII logo and color codes.
  • display.formatter: Formats and colors the output for terminal display.
  • config.default_config: Contains default color settings for keys and values.

Error Handling

  • If an error occurs during execution, it will be displayed in red with a descriptive message.
  • Pressing Ctrl + C safely exits the program without crashing.

Example Output




OS: Helwan Linux 2026
Kernel: 6.18.7-zen
CPU: Intel i7-12700K
RAM: 32 GB (75% used)
Disk: 1 TB (60% free)
Desktop: KDE Plasma 6.6
Network: Connected via Wi-Fi (192.168.1.10)
Quote: "Keep pushing forward, success is near."

Note

Helfetch is designed to work on Helwan Linux but may display partial information on other Linux distributions. Use --no-logo if you want a cleaner terminal output without ASCII art.
