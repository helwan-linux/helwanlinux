🌍 Contributing to Helwan Linux

   

Thank you for your interest in contributing to Helwan Linux — a modern Arch-based Linux distribution crafted for developers, power users, and creators around the world.

   

Helwan Linux is designed to be fast, focused, and friendly — with built-in support for Right-to-Left (RTL) languages like Arabic, Hebrew, and Persian.

   
Not just another distro. Helwan is a developer-first and gamer-ready ecosystem.
   

🚀 What Makes Helwan Linux Unique?

   
         
  • Over 230+ curated packages pre-installed.
    •      
  • Lightweight and customizable Cinnamon environment.
    •      
  • Complete dev stack: Rust, Python, GTK, Qt, Docker, git...
    •      
  • Helwan Native Tools: hel-terminal, hel-store, hel-ai-gate, hel-insight, etc.
    •    
   

🎮 Gaming Support & Performance Tools

   

Pre-installed Games: cuyo, blurble, artikulate, openra

   

Tools: Lutris, VKD3D, GameMode, MangoHud

   

🤝 Who Can Contribute?

   
         
  • Developers: C, Python, Rust, GTK, Qt, Shell...
    •      
  • Translators: RTL/LTR support
    •      
  • Designers: UI, themes, icons
    •    
   

🛠️ How to Contribute

   
         
  1. Fork the GitHub repo: https://github.com/helwan-linux
    2.      
  2. Clone locally and pick a tool (hel-insight, hel-terminal...)
    3.      
  3. Follow README/INSTALL files inside each folder
    4.      
  4. Create a feature branch → push → open Pull Request
    5.    
   

🧭 Contribution Guidelines

   
         
  • Use clear commit messages (e.g. fix:, feat:)
    •      
  • Open an Issue for major ideas
    •      
  • Test both RTL and LTR
    •      
  • Be respectful and collaborative
    •
